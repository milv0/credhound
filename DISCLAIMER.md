## 사용자 책임 (User Responsibilities)

본 도구를 사용함으로써 사용자는 다음에 동의합니다:

1. **권한 있는 사용만 허용**: 본인이 권한을 가진 시스템 및 파일에 대해서만 사용하세요.
2. **결과 검증**: 스캔 결과는 오탐(false positive) 또는 미탐(false negative)을 포함할 수 있습니다. 반드시 수동 검증하세요.
3. **Credential 처리**: 발견된 credential의 적절한 처리(폐기, 교체, 보고)는 사용자 책임입니다.
4. **법령 준수**: 사용자는 적용 가능한 모든 법령을 준수할 책임이 있습니다.
5. **무단 접근 금지**: 타인의 시스템에 대한 무단 스캔은 법적 처벌의 대상이 될 수 있습니다.

## 보안 경고 (Security Notice)

- 본 도구는 자동화된 탐지 도구이며, **보안 전문가의 검토를 대체하지 않습니다**.
- 스캔 결과만으로 보안 결정을 내리지 마세요.
- `--unmask` 옵션 사용 시 실제 credential 값이 리포트에 포함됩니다. 리포트 공유 시 주의하세요.
- 리포트 파일에 민감한 정보가 포함될 수 있으므로 적절히 보호하세요.

## 관련 법령 안내

본 도구 사용 시 다음 법령을 인지하고 준수하세요:

| 법령 | 관련 사항 |
|------|----------|
| 개인정보보호법 (PIPA) | 스캔 결과에 개인정보 포함 시 적용 |

| 부정경쟁방지법 | 영업비밀 해당 credential 탐지 시 적용 |
| GDPR | EU 거주자 데이터 처리 시 적용 |
| CFAA (미국) | 미국 내 사용 시 적용 |
